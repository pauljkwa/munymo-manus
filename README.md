# Munymo - Wordle for Finance

A daily stock prediction game where players guess whether a stock will go up or down.

## Project Overview

Munymo is an MVP (Minimum Viable Product) designed to test the concept of a daily stock prediction game with a Wordle-like experience. Players make one prediction per day, compete on leaderboards, and earn points based on correctness, speed, and streaks.

## Tech Stack

- Frontend: React + TypeScript + TailwindCSS
- Backend: Node.js + Express/Next.js
- Database: PostgreSQL
- Hosting: Manus (MVP), exportable to DigitalOcean

## Third-Party Integrations

- Finnhub: Stock market data
- Stripe: Payment processing
- Firebase: Push notifications
- SendGrid: Email service
- Sentry: Error tracking
- Google Analytics: User analytics
